Attendance Management System

📌 Overview

The Attendance Management System is a web-based application designed to simplify and automate student attendance tracking. The system allows administrators and teachers to manage student records, mark attendance, monitor attendance history, and generate reports efficiently.

This project reduces manual paperwork, improves accuracy, and provides a centralized platform for attendance management.

---

🚀 Features

- Student Registration and Management
- Attendance Marking System
- Attendance History Tracking
- Student Search Functionality
- Dashboard for Attendance Overview
- Secure Data Management
- Responsive User Interface
- Real-Time Attendance Updates

⚙️ Installation

1. Clone the Repository

git clone https://github.com/swainsekhar902-afk/attendance-system.git

2. Navigate to Project Folder

cd attendance-system

3. Install Dependencies

npm install

4. Configure Environment Variables

Create a ".env.local" file and add:

GEMINI_API_KEY=your_gemini_api_key

5. Run the Application

npm run dev

6. Open Browser

http://localhost:3000
